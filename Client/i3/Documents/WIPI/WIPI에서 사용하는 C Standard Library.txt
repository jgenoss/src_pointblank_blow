WIPI에서 사용하는 C Standard Library

작성일: 2005.10.11 
최종수정: 2006.01.04
작성자: 백성민

* 사용 가능한 함수 : memcpy, memset, strcat, strcpy, strcmp, strncmp, strlen, sizeof, ...

* 사용 불가 함수 (폰 다운) : new, delete, malloc, vsprintf, sscanf, strnicmp ...

* WIPI용으로 바꿔 사용해야 할 함수 : sprintf -> MC_knlSprintk (WIPI로 대체해서 사용)
			             printf  -> MC_knlPrintk ( " )
			             fopen   -> MC_fsOpen, 
			             fclose  -> MC_fsClose, 
                   		     fwrite  -> MC_fsWrite, 
                                     fread   -> MC_fsRead,

* 그밖에 주의 사항 : 구조체 사용시 총 사용하는 변수의 크기 4byte 정렬이 되어 있어야 한다.
		     예) INT32  a1;					INT32  a1;
			 UINT8	b1;					UINT8  b1;
 			 UINT8	b2;			===>		UINT8  b2;
			 UINT8	b3;					UINT8  b3;
                             						UINT8  Reverse; 

			총 크기가 7byte 이므로 위험!			8byte 이므로 안전!

* WIPI GIGA SDK를 사용하는 SKT 폰에서는 실수 연산은 FLOATING POINT (부동 소수점)이 안되므로 반드시 
  FIXED POINT (고정 소수점) 연산을 해야 한다. 
  MC_knlPrintk( "%f", fVal); 해도 폰에서는 값은 무조건 0.0만 출력되므로 주의!

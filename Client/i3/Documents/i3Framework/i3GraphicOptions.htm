i3GraphicOptions

 

Game의 Graphic System에 대해 사용자가 지정할 수 있는 Option 정보들을 보관한다.

 

Field

Type

Descriptions

Resolution Width

INT32

Game의 가로 해상도. 일반적으로 몇 가지의 Preset이 주어져야 한다.

Resolution Height

INT32

Game의 세로 해상도. 일반적으로 몇 가지의 Preset이 주어져야 한다.

Refresh Rate

REAL32

Game의 화면 갱신 빈도. Hz로 표현된다

Full-Screen 여부

BOOL

On/Off의 값을 가진다.

Shader 사용 여부

BOOL

On/Off의 값을 가지는 것으로, HW가 Shader를 지원하는 경우에도 Fixed-Pipe Line을 사용하도록 강제할 수 있다.

Texture 해상도 수준

INT32

임의의 값을 가지는 것으로 낮은 값일수록 높은 Quality의 Texture를 사용한다. 그 값의 의미는 Game마다 다를 수 있기 때문에, 0-Level의 Texture가 어떤 Res.가 될지에 대해서는 Game의 개발 시에 별도로 논의되어야 한다.

Anti-Aliasing Level

INT32

Multisampling Quality를 설정하는 항목. 정수형 값을 사용하며, 아래의 의미를 가진다.

 

0

Multisampling을 사용하지 않음.

1~

Multisampling을 적용한다.

Refresh Sync 여부

BOOL

Monitor의 Refresh Rate에 맞추어 화면 갱신을 할 것인지 설정

Mipmap 사용 여부

BOOL

만약 On이라면 Mipmap이 지정된 Texture에 한해 Mipmap을 사용한다. 그렇지 않다면 Mipmap이 있다하더라도 사용하지 않는다.

DXT Format 사용 여부

BOOL

i3Engine 내에서는 DXT Format으로의 압축 및 해제를 Runtime 시에 진행하지 않으며, Game에서 해당 속성에 대한 Resource 관리를 해야 한다.

Gamma

COLOR

R, G, B에 대한 Factor 값을 설정한다.

총 캐릭터 표시 수

INT32

i3Framework 내에서는 캐릭터의 표시 수에 대한 어떠한 자동 처리도 하지 않으며, Game Framework에서 해당 처리를 해야 한다.

Shadow Detail Level

INT32

i3Framework 내에서는 Shadow의 Detail Level에 대해 미리 정해진 바가 없기 때문에, Game Framework에서 해당 처리를 해야 한다.

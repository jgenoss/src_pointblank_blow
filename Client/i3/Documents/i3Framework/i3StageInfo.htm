i3StageInfo

 

i3StageInfo Class로 표현된다. 개별 Stage 정보는 아래와 같은 세부 항목들을 가진다.

Field

Desc.

Class Name

i3Stage Class에서 파생된 Stage Class 명. i3Framework 내부에서 Jump-Stage가 이루어질 때, 지정된 Class Name으로 Class를 생성한다.

Fade-In Color

Fade-In 시에 적용할 화면의 색상

Fade-In Time

Fade-In이 진행되는 시간. 초(sec) 단위로 표현.

Fade-Out Color

Fade-Out 시에 적용할 화면의 색상

Fade-Out Time

Fade-Out이 진행되어, 완전히 화면이 사라질 때까지의 시간. 초(sec) 단위로 표현.

Next Stage

다음 Stage에 대한 Pointer

Jump Type

Next Stage로 Jump할 때의 동작 유형. 아래의 값들을 선택적으로 사용할 수 있다.

　

 Constant

Desc.

I3FRM_STAGE_JUMP_SET

Next Stage로 단순 이전한다.

I3FRM_STAGE_JUMP_PUSH

현재 Stage를 내부 Stage Stack에 넣고 Next Stage로 이전한다. Pop-Jump에 의해, 현재 Stage로 다시 복귀할 수 있다.

I3FRM_STAGE_JUMP_POP

Push-Jump에 의해 전환된 Stage에서 이전 Stage로 복귀할 때 사용한다.

　

　

App.는 전체 Stage Information들을 i3Framework에 제시해야 한다.

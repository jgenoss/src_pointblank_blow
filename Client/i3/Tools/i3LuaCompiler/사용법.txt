실행시 인자값이 필요합니다.

인자1 : 옵션
    - 로그타입 : N (새로쓰기), A (이어쓰기), X (로그안씀)      -> 로그를 안써도 커맨드 창에는 출력됨
    - 덤프 : O (덤프함), X (덤프안함)
    - 디버그정보 : 0 (포함), X (포함안함)
    - 루아파일로드에러출력 : O (출력함), X (출력 안함)         -> 루아파일 코드에러를 알수 있다.
    - 루아파일소스에러출력 : O (출력함), X (출력 안함)
    - 제외 파일 리스트 : O (제외파일 있음), X (없음)

    - 순서대로 입력해야함. 하나라도 누락되면 안됨. 아예 생략은 가능. 대소문자 상관없음

    - ex) NOOOO   -> 로그새로쓰고 덤프하고 디버그정보포함에 루아파일 로드랑 소스에러 출력해줌
    - 숫자로 옵션 선택가능 1~4
        = 1 : NOXOXX
        = 2 : AOXOXX
        = 3 : NOXOX0
        = 4 : AOXOX0
        = 5 : NXXOXX      -> 해당옵션으로 루아파일 오류만 알아내는데 사용가능
        = 6 : AXXOXX


인자2 : 경로&필터	- ex) *.*    UIRe/Script/*.lua
인자3 : 출력확장자	- ex) .luab
인자4 : 제외파일리스트
    - 제외파일은 ;로 구분
    - 제외파일리스트 옵션 X일때 생략 가능
    - ex) "a.a;b.b;"

툴은 빌드시 i3/Bin 폴더에 복사 됩니다.
